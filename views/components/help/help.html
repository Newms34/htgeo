Description

Welcome to the Brethren [PAIN] (henceforth known as “Brethren”, because writing that guild tag stuff is annoying) Guild Wars 2 Guild website! This is basically just a tool I (Dave Newman/Portable Healy Unit) wrote to make organizing guild stuff easier. The site has a bunch of sections that you may find useful.

Sections

There are five major sections of the site:

  • The Dashboard: View/edit your personal information (including characters), send and receive messages, look at other members’ profiles, and see upcoming events
  • Calendar: View and create/edit events. This is still a bit of a WIP, but it allows/will allow Brethren members to create events on a nice, convenient calendar.
  • Chat: You can chat. The chat here is not preserved (i.e., I cannot retrieve your week old conversation), but still, be nice.
  • Forums: Got a message you want everyone to see? Post it here. You can even include (small) pictures! As always, be nice. I’ve included an voting system for each post (yes, like reddit), so feel free to tell each other how stupid/awesome you think your ideas are
  • Help: You’re looking at it. Hi!

Accounts

Before using the site (and actually, viewing this page), you’re gonna need to create a account. You may note that creating an account involves just creating a username and passsword, and not any other information. This is because while that other info (i.e., your characters, preferred game mode, etc.) is cool, I don’t want to force you to make it before you sign up.

Confirming

Once you’ve created your account, you’ll need it to be confirmed. This is so that while anyone can create an account, only users confirmed by Brethren site moderators can actually view the site. Trolls r bad, k?
